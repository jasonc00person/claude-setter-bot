# Notion CRM Process — Step-by-Step

> For CRM schema, properties, and status mapping, see `CLAUDE.md` (Notion CRM Integration section).
> For Notion keyboard shortcuts, see `memory/notion-ui-shortcuts.md`.

## Quick Access

- **URL:** `https://www.notion.so/creator-economy/27ef65855e6b8032b52ef6b507c12144`
- **Default view:** Board (grouped by Lead Status)
- **Alt view:** Table (for bulk editing)

## How to Add a New Lead

1. Navigate to CRM Board view
2. Find the correct status column (e.g. "Booking Link Sent")
3. Click "+ New page" at the bottom of that column
4. Type the lead's name and press Enter
5. Card appears — hover over it and click the expand/open icon (small square icon)
6. On the card page, click "Empty" next to "IG Profile Link"
7. Type the profile URL: `https://www.instagram.com/[handle]/`
8. Press Enter to save
9. Fill any other known fields (Country, etc.)

## How to Update an Existing Lead

1. Find them on the Board view (scan each column)
2. Click their card name to open
3. To change status: click the current status tag, select new status from dropdown
4. To add IG Profile Link: click "Empty" next to "IG Profile Link", type URL, Enter

## When to Create CRM Entries

- **Skool members:** Should already exist (pre-populated)
- **IG-only leads:** Create a new entry when they advance **past Stage 2**
- **Don't create entries for:** Stage 1 leads, ManyChat auto-leads who haven't engaged

## Editing Fields in Notion (CRITICAL)

- **To edit any field value:** Click RIGHT NEXT TO the text, but NOT directly ON the text itself. Clicking on the text (especially URLs) will navigate to it or open it instead of editing.
- For URL fields with existing values: click in the empty space beside the URL text to open the edit input
- **Verify column before entering data** — Lead Status and IG Profile Link are adjacent, easy to mix up
