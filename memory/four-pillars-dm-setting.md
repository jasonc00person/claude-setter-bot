If you're not booking calls, you're missing one of these 4 things. Every session must map to at least one, and the daily flow should cycle through all 4.

## The 4 Pillars

1. **Conversation** — Handle replies. Move active leads through the pipeline stages. This is where calls get booked.
2. **Follow-ups** — Re-engage leads who went quiet. Nudge "Awaiting Booking" leads. Check on Stage 2 no-replies. Bring stale leads back to life.
3. **Prospecting** — Find new ICP leads. Check IG notifications (story viewers, likers, new followers). Browse Skool members. Build the top of the funnel.
4. **Outreach** — Send personalized Stage 1 openers to the prospects you found. This seeds tomorrow's conversations.

## Recommended Daily Order

| Order | Pillar | Why This Order |
|-|-|-|
| 1st | **Conversation** | Handle replies first. These are warmest. Booking calls = highest ROI activity |
| 2nd | **Follow-ups** | Re-engage stale leads while energy is high. These are warmer than cold outreach |
| 3rd | **Prospecting** | Find new ICP leads from notifications, Skool, etc. Build the list |
| 4th | **Outreach** | Send openers last so leads see them overnight and reply by next session |

## How This Maps to Existing Workflows

- **ManyChat workflow** = primarily Conversation (replies) + Follow-ups
- **Skool workflow** = Conversation (unread chats) + Follow-ups + Outreach (new member openers)
- **Instagram Notifications workflow** = Prospecting + Outreach only
- **A full day** should touch all 4 across these workflows

## Why This Matters

**Why:** Jason identified that when call bookings slow down, it's always because one of these 4 pillars is being neglected. They're a closed loop — outreach seeds conversations, conversations need follow-ups, follow-ups need fresh prospects to replace drop-offs.

**How to apply:** At session start, identify which pillar(s) this session covers. At session end, note which pillars were NOT touched today so the next session can prioritize them. Never go 2 days without hitting all 4.
