# Skool DM Setting Workflow

## Overview

Skool is a secondary DM setting channel alongside Instagram/ManyChat. Two groups:
- **Creator Accelerator (paid):** skool.com/creatoraccelerator — 222 members, $67-75/mo
- **Internet Money Central (free):** skool.com/internet-money-central-9086 — ~2K members

## Priority Order

1. **Respond to people** — Check Skool chat (chat icon top-right), handle all unread (blue dots) first
2. **Outreach current CA members** — Go to Members tab, work through newest members who haven't been messaged
3. **Outreach free group members** — Internet Money Central, same approach but lower priority
4. **Draft promotional post for free group** — Toggle "send email to all" for max reach

## Navigation

- **Chat panel:** Click chat icon (speech bubble) in top-right nav bar
- **Filter chats:** Dropdown next to "Mark all as read" — toggle between "All" and "Unread"
- **Members page:** Members tab in nav, or `/creatoraccelerator/-/members`
- **Member chat:** Click "CHAT" button on member card, or click their name to open chat overlay
- **Close chat overlay:** Click X in top-right of chat window

## Member Info Available

Each member card shows:
- Bio/description
- Active status (e.g., "Active 4h ago")
- Join date
- Membership tier ($67/mo, $75/mo, or Free)
- Renewal timeline
- Join source (YouTube, email invite, direct)
- Location (city)

## Skool DM Setting Flow (Adapted from IG Pipeline)

### For New Members (just joined, no prior chat)
1. **Welcome message:** "yooo welcome in bro! glad to have u here"
2. **Personalized opener based on their bio:** Reference what they do, ask about their goal/situation
3. This counts as **Stage 1** — opener sent

### For Members Who Reply
Follow the same 6-stage pipeline as IG:
- Stage 1 -> 2: They reply. Ask AV question (making money or looking to?)
- Stage 2 -> 3: Positive AV. Ask permission to qualify
- Stage 3 -> 4: Permission granted. 3 qualifying questions (goal, challenge, what they've tried)
- Stage 4 -> 5: Qualified. Bridge to call
- Stage 5 -> 6: Agrees to call. Send Calendly link
- Stage 6: Booked. "Sick dude speak soon!"

### For Existing Members Not Yet Messaged
- Check if there's an existing chat (look for prior messages)
- If no prior chat: send welcome + opener based on bio
- If old chat exists but went cold: re-engage with context (reference the $10k/mo post, group calls, etc.)

## Skool-Specific Notes

- **Spam risk warnings:** Skool shows "Spam risk score: High" for some users (often false positives for international members). Ignore unless clearly a bot.
- **Chat URLs:** Can't reliably extract Skool chat URLs. Search by name to find conversations.
- **No ManyChat tags on Skool.** Pipeline-state.md is the only tracker.
- **Platform column:** Use "Skool" in pipeline-state.md for Skool leads.
- **Handle column:** Use Skool @username or "—" if not relevant.

## Free Group Outreach (Internet Money Central)

- URL: skool.com/internet-money-central-9086
- ~2K members — large pool for outreach
- Same welcome + opener approach
- These are colder leads (free group) so expect lower conversion
- Goal: Get them interested in paid CA or onto a call

## Promotional Post (Lowest Priority)

- Draft a value-driven post for the free group
- Toggle "send email to all" to notify all members
- Post should tease value from CA or a specific resource
- Drive engagement -> DM conversations -> pipeline

|-|-|-|
